# 打包步骤

## 1. 安装PyInstaller

```bash
pip install pyinstaller
```

## 2. 运行打包脚本

```bash
python build_exe.py
```

## 3. 生成的文件

打包完成后，在 `dist` 目录下找到：
- `媒体摘要PPT生成器.exe` - 主程序（约80-100MB）

## 4. 分发文件结构

创建以下文件夹结构用于分发：

```
媒体摘要PPT生成器/
├── 媒体摘要PPT生成器.exe
├── example/
│   └── 2026年1月媒体摘录.xlsx
├── 使用说明书.md
└── output/            # 输出目录（可选，程序会自动创建）
```

## 注意事项

1. **ChromeDriver**：
   - exe不包含ChromeDriver
   - 用户需要安装Chrome浏览器
   - 程序运行时会自动使用系统的Chrome

2. **文件大小**：
   - 单文件exe约80-100MB
   - 包含所有Python依赖

3. **运行要求**：
   - Windows 10/11
   - 已安装Chrome浏览器
   - 网络连接

4. **测试**：
   - 打包完成后，先在另一台没有Python的电脑上测试
   - 确保所有功能正常
