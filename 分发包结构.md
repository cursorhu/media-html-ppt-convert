# 分发包结构

## 📦 给用户的完整包

将以下文件和文件夹打包给用户：

```
媒体摘要PPT生成器/
├── 媒体摘要PPT生成器.exe    # 主程序（217MB）
├── input/                    # 输入文件夹（必须创建）
│   └── 媒体摘录.xlsx        # 示例Excel文件（用户替换为自己的）
├── exe使用说明.md            # 使用说明
└── config.py                # 配置文件（可选，用于自定义路径）
```

## 📋 准备步骤

### 1. 创建分发文件夹

```bash
mkdir 媒体摘要PPT生成器
cd 媒体摘要PPT生成器
```

### 2. 复制文件

- 从 `dist/` 复制 `媒体摘要PPT生成器.exe`
- 复制 `exe使用说明.md`
- 复制 `config.py`（可选）

### 3. 创建input文件夹并放入示例

```bash
mkdir input
# 将 example/2026年1月媒体摘录.xlsx 复制到 input/媒体摘录.xlsx
```

### 4. 打包压缩

将整个 `媒体摘要PPT生成器/` 文件夹压缩为zip

## 🚀 用户使用流程

1. 解压zip文件
2. 将自己的Excel文件放入 `input/` 文件夹，命名为 `媒体摘录.xlsx`
3. 运行exe：
   ```bash
   # 测试模式
   媒体摘要PPT生成器.exe --test
   
   # 完整处理
   媒体摘要PPT生成器.exe
   ```
4. 在 `output/` 文件夹查看生成的PPT

## ⚠️ 重要说明

### 路径结构

- ✅ **input/**：用户必须创建此文件夹，放入Excel文件
- ✅ **output/**：程序自动创建，存放生成的PPT
- ✅ **temp/**：程序自动创建，临时文件（自动清理）

### Excel文件要求

- 文件名：`媒体摘录.xlsx`（默认）
- 位置：`input/媒体摘录.xlsx`
- 格式：第一行表头，第二行开始数据

### 自定义Excel路径

如果用户想使用不同的文件名或路径，修改 `config.py`：

```python
EXCEL_PATH = os.path.join(BASE_DIR, 'input', '你的文件名.xlsx')
# 或绝对路径
EXCEL_PATH = r"D:\my\path\媒体摘录.xlsx"
```
