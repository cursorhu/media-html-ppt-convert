# 快速开始指南

## 测试运行（推荐先测试）

```bash
python main.py --test
```

这将处理前5行数据，验证功能是否正常。

## 处理完整数据

```bash
python main.py
```

这将处理Excel中的所有150行数据，生成完整的PPT。

## 输出文件

生成的PPT位于：`output/媒体摘录.pptx`

## 注意事项

1. **网络连接**：需要稳定的网络访问CCTV网站
2. **处理时间**：每行数据约需20-30秒（包括页面加载和截图）
3. **ChromeDriver**：首次运行时会自动下载ChromeDriver
4. **错误处理**：单条数据失败不会影响整体流程，会继续处理下一条

## 查看日志

如果遇到问题，查看详细日志：

```bash
type process.log
```

## 自定义配置

编辑 `config.py` 可以调整：
- 页面加载等待时间
- 图片尺寸
- 摘要长度
- 超时和重试设置
