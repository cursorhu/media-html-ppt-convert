# 最终使用说明

## 🎯 最简使用方式

### 1. 准备Excel文件
将你的Excel文件（任意名称）放入 `input` 文件夹：

```
媒体摘要PPT生成器.exe
input/
  └── 你的Excel文件.xlsx  (任意文件名)
```

### 2. 运行程序
直接运行exe，程序会自动找到Excel文件：

```bash
媒体摘要PPT生成器.exe
```

**自动识别规则**：
- ✅ 自动扫描input目录下的所有.xlsx和.xls文件
- ✅ 如果有多个文件，使用第一个
- ✅ 如果input目录没有，自动查找example目录
- ✅ 不需要指定文件名！

### 示例1：最简单（推荐）

```bash
# 1. 将Excel文件放入input文件夹

```

## ❓ 常见问题

### Q: 如果有多个Excel文件怎么办？
A: 程序会使用第一个文件，并在日志中提示哪些文件被忽略

### Q: 支持什么格式？
A: 支持.xlsx和.xls格式

### Q: 必须命名为"媒体摘录.xlsx"吗？
A: **不需要**！可以是任意文件名

### Q: 没有input文件夹怎么办？
A: 程序会自动创建input文件夹并提示
