# 更新日志

## 2026-06-12 更新

### ✅ 新增功能

1. **链接显示**
   - PPT文字部分现在包含链接URL
   - 链接显示为蓝色字体，易于识别

2. **改进测试模式**
   - `--test` 模式现在包含视频和图文两种类型
   - 测试数据：3条视频 + 2条图文（人民日报、中国新闻网）
   - 更全面地验证系统功能

### 📊 PPT布局

每页幻灯片包含：
- **标题栏**：媒体名称 + 新闻标题
- **信息栏**：
  - 发布时间
  - 链接URL（蓝色显示）
- **内容区**：
  - 视频：视频播放器截图
  - 图文：网页截图 + 文本摘要（300字）

### 🔧 技术改进

- 优化图文网页的文本提取
- 改进PPT排版，为链接预留空间
- 测试模式使用独立输出文件，避免文件冲突

### 🧪 测试结果

测试模式（5条数据）：
- ✅ 3条CCTV视频链接 - 成功
- ✅ 1条人民日报图文 - 成功，提取303字摘要
- ✅ 1条中国新闻网图文 - 成功，提取303字摘要
- 平均处理时间：6-9秒/条
- 总耗时：约33秒

### 📝 使用方法

```bash
# 测试模式（3视频+2图文）
python main.py --test

# 处理所有150行数据
python main.py

# 处理指定行数
python main.py --limit 20
```
