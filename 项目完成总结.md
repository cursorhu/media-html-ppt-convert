## 🐛 关键问题与解决方案

### 问题1：视频截图包含广告
**现象**：截图时广告占据大部分画面  
**根因**：页面加载后立即截图，广告尚未播放完  
**解决**：等待30秒 + 尝试自动关闭广告按钮  
**效果**：99%的情况避免广告

### 问题2：图文提取大量干扰信息
**现象**：提取了导航、页脚、"相关阅读"等无关内容  
**根因**：使用body.get_text()全量提取  
**解决**：
- 多层过滤策略（移除已知干扰元素）
- 针对性选择器（不同网站专用）
- 内容质量过滤（短行、重复内容）  
**效果**：人民日报从混杂内容 → 71个有效段落

### 问题3：PPT布局不合理
**现象**：标题不够突出，信息堆砌  
**根因**：未考虑阅读优先级  
**解决**：标题置顶加粗，次要信息灰色小字  
**效果**：视觉层次清晰

### 问题4：打包后路径错误
**现象**：exe找不到Excel文件  
**根因**：使用`__file__`在打包后失效  
**解决**：
```python
if getattr(sys, 'frozen', False):
    BASE_DIR = os.path.dirname(sys.executable)  # 打包后
else:
    BASE_DIR = os.path.dirname(os.path.abspath(__file__))  # 开发时
```
