**修复前**：
```
媒体名称 + 发布时间  ← 在顶部
标题               ← 在下方
```

**修复后**：

### 2. 视频截图避免广告 ✅

**优化策略**：

1. **延长等待时间**
   - 基础页面加载：3秒
   - 视频页面额外等待：5秒（让广告播放完毕）
   - 总计：8秒等待时间

2. **尝试关闭广告**
   - 自动检测并点击关闭按钮
   - 支持多种关闭按钮选择器：`.close`, `.closeBtn`, `.ad-close`

3. **截图逻辑**
   - 优先截取视频播放器元素（`video`, `.player`）
   - 降级方案：整页截图
