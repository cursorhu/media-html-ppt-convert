# exe使用说明

## 📁 文件结构

将exe和相关文件按以下结构组织：

```
媒体摘要PPT生成器/
├── 媒体摘要PPT生成器.exe    # 主程序
├── input/                    # 输入文件夹（新建）
│   └── 媒体摘录.xlsx        # 将你的Excel文件放这里
├── output/                   # 输出文件夹（程序自动创建）
│   └── 媒体摘录.pptx        # 生成的PPT
├── temp/                     # 临时文件夹（程序自动创建）
└── process.log              # 日志文件（程序自动创建）
```

## 🚀 使用步骤

### 1. 准备Excel文件

将你的Excel文件命名为 `媒体摘录.xlsx`，放在 `input` 文件夹中。

**Excel格式要求**：
- 第一行是表头：序号、媒体名称、发布时间、标题、链接、备注
- 第二行开始是数据

### 2. 运行程序

**命令行方式（推荐）**：

```bash
# 测试模式（3视频+2图文）
媒体摘要PPT生成器.exe --test

# 处理所有数据
媒体摘要PPT生成器.exe

# 处理指定行数
媒体摘要PPT生成器.exe --limit 20
```

**双击运行**：
- 直接双击exe，程序会处理所有数据

### 3. 查看结果

处理完成后，在 `output` 文件夹中找到生成的PPT文件。

## ⚙️ 自定义Excel路径

如果你的Excel文件不在默认位置，可以修改 `config.py`：

```python
EXCEL_PATH = r"D:\your\path\媒体摘录.xlsx"
```

将 `config.py` 放在exe同目录下即可。

## 📝 注意事项

1. **input文件夹**：必须手动创建，并放入Excel文件
2. **Excel文件名**：默认是 `媒体摘录.xlsx`，可以在config.py中修改
3. **output文件夹**：程序会自动创建
4. **Chrome浏览器**：必须安装

## ❓ 常见问题

**问题1：提示找不到Excel文件**

解决方法：
1. 检查是否创建了 `input` 文件夹
2. 检查Excel文件名是否为 `媒体摘录.xlsx`
3. 检查文件路径是否正确

**问题2：想使用不同的Excel文件**

解决方法：
1. 重命名你的文件为 `媒体摘录.xlsx`
2. 或者创建 `config.py` 修改 `EXCEL_PATH`
