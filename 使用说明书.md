# 媒体摘要PPT生成器 - 完整使用指南

## 📦 工具简介

本工具可以自动将Excel中的媒体摘录数据转化为精美的PPT演示文稿。

**核心功能**：
- 📊 **自动识别Excel文件** - 零配置，放入input文件夹即可
- 🎬 **视频智能截取** - 自动等待广告播放完毕后截取画面
- 📄 **图文智能提取** - 多层过滤算法，精准提取正文内容
- 📑 **PPT自动生成** - 标准化排版，包含标题、媒体信息、链接、截图和摘要

## 🎯 快速开始（3步上手）

### 第1步：准备Excel文件

将你的Excel文件放入 `input` 文件夹（任意文件名）：

```
媒体摘要PPT生成器.exe
input/
  └── 你的文件.xlsx  ← 任意名称
```

**Excel格式要求**：

| 序号 | 媒体名称 | 发布时间 | 标题 | 链接 | 备注 |
|------|---------|---------|------|------|------|
| 1 | CCTV-13 | 2026-01-01 | 新闻标题 | http://... | 可选 |
| 2 | 人民日报 | 2026-01-02 | 文章标题 | http://... | 可选 |

### 第2步：运行程序

**最简方式（推荐）**：
```bash
媒体摘要PPT生成器.exe
```

**测试模式（首次建议）**：
```bash
媒体摘要PPT生成器.exe --test
```

### 第3步：查看结果

生成的PPT文件位于 `output` 文件夹：
- 完整模式：`output/媒体摘录.pptx`
- 测试模式：`output/测试_媒体摘录.pptx`

## ⚙️ 核心特性

### ✅ 自动识别Excel文件
- 自动扫描input目录下的所有.xlsx和.xls文件
- 无需指定文件名，零配置即可使用
- 多个文件时自动使用第一个并记录日志

### 🎬 智能视频截取
- **30秒广告等待**：自动等待视频广告播放完毕
- **广告按钮检测**：主动查找并点击关闭按钮
- **画面完整性**：确保截取的是视频内容，不含广告

### 📄 智能图文提取
- **4层过滤算法**：
  1. 优先识别文章容器（article, .content, #artibody等）
  2. 过滤导航、侧边栏、页脚（nav, aside, footer等）
  3. 网站特定选择器（人民日报.left_zw、新浪.artibody等）
  4. 智能密度分析（保留文字密度高的段落）
- **噪音过滤**：自动排除"分享到"、"责任编辑"、"相关阅读"等干扰信息

### 📑 标准化排版
- **一致的视觉风格**：统一字体、颜色、间距
- **自适应图片**：自动缩放适配幻灯片尺寸
- **清晰的信息层级**：标题 > 媒体信息 > 链接 > 内容

## ⏱️ 处理时间参考

| 数据量 | 包含视频 | 全图文 | 说明 |
|--------|---------|--------|------|
| 5条（测试） | ~3分钟 | ~1分钟 | 3视频+2图文 |
| 20条 | ~12分钟 | ~4分钟 | 视频需等待广告 |
| 50条 | ~30分钟 | ~10分钟 | 建议分批处理 |
| 150条 | ~90分钟 | ~30分钟 | 可用--limit分批 |

**时间构成**：
- 视频链接：~33秒/条（30秒广告 + 3秒加载）
- 图文链接：~5-8秒/条（取决于网络和页面复杂度）

## ⚙️ 高级配置

如需调整参数，编辑 `config.py`：

```python
# 视频广告等待时间（秒）
VIDEO_AD_WAIT = 30  # 广告长时改为60

# 页面加载等待时间（秒）
PAGE_LOAD_WAIT = 3

# 摘要最大长度（字符）
SUMMARY_MAX_LENGTH = 300

# 网络超时（秒）
REQUEST_TIMEOUT = 30
```

## ❓ 常见问题

### 1. 找不到Excel文件

**错误信息**：`未在 input 目录找到Excel文件`

**解决方法**：
```bash
# 方案1：创建input文件夹并放入Excel
mkdir input
copy 你的文件.xlsx input/

# 方案2：直接指定文件路径
媒体摘要PPT生成器.exe --excel "D:\data\文件.xlsx"
```

### 2. 视频截图包含广告

**原因**：视频广告超过30秒

**解决方法**：
1. 编辑 `config.py`
2. 将 `VIDEO_AD_WAIT = 30` 改为 `VIDEO_AD_WAIT = 60`
3. 重新打包：`python build_exe.py`

### 3. 图文摘要包含噪音

**原因**：网站结构特殊，未被识别

**解决方法**：
1. 查看日志 `process.log` 定位URL
2. 在 `webpage_scraper.py` 的 `_get_site_specific_selectors()` 中添加该网站的选择器
3. 重新打包

### 4. Chrome浏览器未安装

**错误信息**：找不到ChromeDriver

**解决方法**：
- 安装Chrome浏览器：https://www.google.cn/chrome/
- 重启程序

### 5. 处理速度太慢

**优化方案**：
```bash
# 分批处理
媒体摘要PPT生成器.exe --limit 30

# 或先测试效果
媒体摘要PPT生成器.exe --test
```

## 🎓 进阶使用
