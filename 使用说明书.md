# 媒体摘要PPT生成器 - 使用说明书

## 📦 工具简介

本工具可以自动将Excel中的媒体摘录数据转化为精美的PPT演示文稿。

**主要功能**：
- 📊 读取Excel媒体摘录数据
- 🎬 自动识别并截取视频画面
- 📄 自动抓取图文网页并提取摘要
- 📑 生成排版美观的PPT文件

---

## 🚀 快速开始

### 方式一：使用打包的exe（推荐）

#### 1. 准备工作

**下载并解压工具包**，确保包含以下文件：
```
媒体摘要PPT生成器/
├── 媒体摘要PPT生成器.exe    # 主程序
├── example/                  # 示例文件夹
│   └── 2026年1月媒体摘录.xlsx
└── 使用说明书.md             # 本文件
```

**安装Chrome浏览器**（如果还没有）：
- 下载地址：https://www.google.cn/chrome/

#### 2. 准备Excel文件

将您的Excel文件放在任意位置，确保格式如下：

| 序号 | 媒体名称 | 发布时间 | 标题 | 链接 | 备注 |
|------|---------|---------|------|------|------|
| 1 | CCTV-13 | 2026-01-01 | 新闻标题 | http://... | 备注信息 |
| 2 | 人民日报 | 2026-01-02 | 文章标题 | http://... | 备注信息 |

**注意**：
- 第一行是表头（序号、媒体名称等）
- 链接列必须填写完整的网址
- 支持视频链接和图文链接

#### 3. 运行程序

**方法A：双击exe文件**
1. 双击 `媒体摘要PPT生成器.exe`
2. 按提示输入Excel文件路径
3. 等待处理完成
4. 在 `output` 文件夹查看生成的PPT

**方法B：命令行运行（推荐）**

打开命令提示符（cmd）或PowerShell，进入工具目录：

```bash
# 测试模式（处理前5条数据）
媒体摘要PPT生成器.exe --test

# 处理所有数据
媒体摘要PPT生成器.exe

# 处理指定行数
媒体摘要PPT生成器.exe --limit 20
```

#### 4. 查看结果

处理完成后，在 `output` 文件夹中找到生成的PPT文件：
- **测试模式**：`测试_媒体摘录.pptx`
- **完整模式**：`媒体摘录.pptx`

## ❓ 常见问题

### 1. 提示"Chrome浏览器未安装"

**解决方法**：
- 安装Chrome浏览器：https://www.google.cn/chrome/
- 重启程序

### 2. 部分页面抓取失败

**原因**：
- 网络连接不稳定
- 网站反爬虫限制
- 页面结构变化

**解决方法**：
- 检查网络连接
- 重新运行程序（会跳过失败的条目）
- 查看 `process.log` 了解详细错误

### 3. 视频截图包含广告

**原因**：视频广告超过30秒

**解决方法**：
- 修改 `config.py` 中的 `VIDEO_AD_WAIT` 参数（如改为60秒）
- 重新运行程序

### 4. Excel文件格式错误

**确保**：
- Excel文件第一行是表头
- 列名正确：序号、媒体名称、发布时间、标题、链接、备注
- 链接列包含完整URL
- 文件未损坏

### 5. 生成的PPT无法打开

**解决方法**：
- 确保安装了Microsoft Office或WPS
- 检查输出文件是否完整（不是0字节）
- 查看日志文件 `process.log`

## 📝 注意事项

1. **网络要求**：需要稳定的互联网连接
2. **浏览器要求**：必须安装Chrome浏览器
